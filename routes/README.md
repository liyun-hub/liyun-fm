# Routes 目录

该目录包含 FM 收音机播放器应用程序的所有路由定义。

## 概述

routes 目录包含应用程序的所有路由定义，用于将 HTTP 请求映射到控制器方法。

## 结构

```
routes/
├── api.php            # API 路由
├── channels.php       # 广播频道
├── console.php        # 控制台路由
└── web.php            # Web 路由
```

## 核心路由

- **web.php**: 定义 Web 应用程序的路由，包括页面访问、表单提交等
- **api.php**: 定义 API 路由，用于提供给前端或其他应用程序调用
- **channels.php**: 定义广播频道，用于实时通信
- **console.php**: 定义控制台命令路由

## 用途

该目录用于组织应用程序的所有路由，使请求能够正确地映射到对应的处理方法，是应用程序的入口点之一。
