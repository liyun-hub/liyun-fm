# Database 目录

该目录包含 FM 收音机播放器应用程序的数据库相关文件。

## 概述

database 目录包含数据库迁移、种子和工厂文件，用于管理应用程序的数据库结构和测试数据。

## 结构

```
database/
├── factories/         # 模型工厂
├── migrations/        # 数据库迁移
├── seeds/             # 数据库种子
└── schema/            # 数据库模式（可选）
```

## 核心组件

- **migrations/**: 包含数据库表结构的迁移文件，用于创建和修改数据库表
- **seeds/**: 包含用于填充数据库的种子数据
- **factories/**: 包含用于生成测试数据的模型工厂

## 用途

该目录用于管理应用程序的数据库结构和数据，支持版本控制数据库变更，并提供测试数据生成功能。
